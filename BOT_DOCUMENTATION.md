## 🏗️ Архитектура

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Telegram  │────▶│   Bot.py    │────▶│ AssemblyAI  │────▶│  DeepSeek   │
│  (голос/текст)│     │  (обработка) │     │(транскрибация)│     │(редактирование)│
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                                                │                    │
                                                ▼                    ▼
                                         .ogg → .wav           Результат
                                         (pydub/ffmpeg)        (профессиональный текст)
```

**Поток обработки:**

1. Пользователь выбирает сценарий (Email, Чат, Документация и т.д.)
2. Отправляет голосовое или текст
3. Голосовое → транскрибация (AssemblyAI) → редактирование (DeepSeek)
4. Текст → сразу редактирование (DeepSeek)
5. Результат отправляется пользователю

## 🔑 API и ключи

### Переменные окружения (.env)

```bash
TELEGRAM_BOT_TOKEN="your-telegram-bot-token"
DEEPSEEK_API_KEY="your-deepseek-api-key"
ASSEMBLYAI_API_KEY="your-assemblyai-api-key"
ADMIN_ID="your-telegram-user-id"  # Опционально
```

### Используемые API

| API                  | Назначение                     | Модель/Endpoint                    |
| -------------------- | ------------------------------ | ---------------------------------- |
| **Telegram Bot API** | Взаимодействие с пользователем | Long Polling                       |
| **AssemblyAI**       | Транскрибация голосовых        | `language_code="ru"`               |
| **DeepSeek**         | Редактирование текста          | `deepseek-chat`, `temperature=0.7` |

### Библиотеки

```bash
python-telegram-bot>=20.0   # Telegram Bot API
openai>=1.0.0               # DeepSeek (OpenAI-compatible)
assemblyai                  # Транскрибация
pydub                       # Аудио конвертация
python-dotenv               # Переменные окружения
ffmpeg                      # Системная зависимость
```

## 🗄️ База данных

### Таблицы

#### `access_codes`

| Поле               | Тип       | Описание                 |
| ------------------ | --------- | ------------------------ |
| `id`               | INTEGER   | Primary key              |
| `code`             | TEXT      | Код доступа (уникальный) |
| `telegram_user_id` | INTEGER   | Telegram ID пользователя |
| `assigned_at`      | TIMESTAMP | Время активации          |
| `is_active`        | BOOLEAN   | Активен ли код           |
| `preferred_style`  | TEXT      | Предпочитаемый стиль     |

#### `usage_logs`

| Поле               | Тип     | Описание                 |
| ------------------ | ------- | ------------------------ |
| `id`               | INTEGER | Primary key              |
| `access_code_id`   | INTEGER | FK к access\_codes       |
| `telegram_user_id` | INTEGER | Telegram ID              |
| `message_type`     | TEXT    | "voice" или "text"       |
| `audio_duration`   | REAL    | Длительность аудио (сек) |
| `text_characters`  | INTEGER | Длина текста             |
| `processing_time`  | REAL    | Время обработки (сек)    |

### Авто-создание пользователей (Open Access)

Новые пользователи автоматически получают `auto_{user_id}_{timestamp}` код при первом использовании.

## 🚨 Troubleshooting

### Бот не отвечает

```bash
# Проверить, запущен ли бот
ssh root@80.249.148.167 "ps aux | grep bot"

# Посмотреть логи
tmux attach -t bot
```

### Ошибка 409 Conflict

**Причина:** Несколько инстансов бота одновременно

**Решение:**

```bash
# Убить все процессы бота
pkill -f bot_v2.py

# Перезапустить
tmux send-keys -t bot 'python bot_v2.py' Enter
```

### Нет venv на сервере

```bash
cd ~/voice_bot/voice_assistant_bot
python3 -m venv venv
source venv/bin/activate
pip install python-telegram-bot openai assemblyai pydub python-dotenv
```
